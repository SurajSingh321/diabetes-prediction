# Diabetes Prediction

A simple ML project to predict whether a person is diabetic or not based on medical data.

I built this using Logistic Regression on the Pima Indians Diabetes dataset from Kaggle.

## What it does

Takes medical inputs like Glucose, BMI, Age, Blood Pressure etc. and predicts whether the person is diabetic or not.

## Tech used

- Python
- Scikit-learn (Logistic Regression, StandardScaler)
- Pandas, NumPy

## How to run

pip install pandas scikit-learn numpy
python diabetes_prediction.py

Make sure diabetes.csv is in the same folder.

## Dataset

Pima Indians Diabetes Database from Kaggle — 768 samples.

## Accuracy

Got around 74% accuracy on the test set.
